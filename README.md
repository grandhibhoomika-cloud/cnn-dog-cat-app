🐶🐱 Dog vs Cat Image Classifier (CNN + Streamlit)

This project is a deep learning-based image classification system that predicts whether an image is a dog or a cat. It is built using Convolutional Neural Networks (CNN) with TensorFlow and deployed using Streamlit for a simple web interface.

## 🚀 Features
- CNN model trained on image dataset
- Binary classification (Dog vs Cat)
- Streamlit web app for image upload and prediction
- Displays prediction with confidence score

## 🧠 How it works
1. Upload image in web app
2. Image is resized and normalized
3. CNN model predicts class
4. Output shown as Dog or Cat with confidence score

## ⚙️ Tech Stack
- Python
- TensorFlow / Keras
- CNN (Deep Learning)
- Streamlit
- NumPy
